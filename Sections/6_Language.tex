\section{Language and Referencing Guide}

This document uses several tools to ensure consistent and high\-/quality language and references. This section provides guidance on correctly referencing sections, figures, code, and literature, as well as on proper punctuation, quotation, and list formatting practices.

\subsection{Referencing}

Cross\-/referencing is done using the \texttt{cleveref} package, which automatically includes the type of reference (e.g., figure, algorithm, listing) in the citation. For example, \cref{lst:fibonacci} shows how to calculate Fibonacci numbers while \cref{fig:theway} displays an image. Algorithms and procedures are referenced similarly, such as \cref{alg:expBySquaring} and \cref{pro:mult}. \Cref{fig:theway,fig:subfigures,lst:csharpMath,lst:mathMinted,lst:matlabHelloWorld,tab:budget,tab:standard}. \texttt{cleveref} does not order these (figures then listings then tables), so you need to reference them in alphabetical order yourself.

The \texttt{hyperref} package enables clickable links throughout the document, including cross\-/references, citations, and the table of contents. The \texttt{bookmark} package extends this with improved PDF bookmark generation. The \texttt{refcheck} package (loaded with the \texttt{norefs} option) warns about unused labels during debugging --- remove \texttt{norefs} to enable these warnings.

References to academic works should be made using \verb|\cite| or \verb|\textcite|. For example, \cite{xx00} discusses some of the concepts used here while \textcite{xx00} provides an in\-/depth explanation. The difference is that \verb|\textcite| integrates the citation naturally into the sentence structure.

\subsection{Hyphenation and Punctuation}

Different dashes are used for different purposes in English typesetting:
\begin{itemize}
    \item \textbf{Hyphen (\-/)} is used to join words, as in \enquote{well\-/known algorithm}.
    \item \textbf{En dash (\-/\-/)} is used for numeric ranges, such as \enquote{pages 10\--20} or \enquote{January\--March}.
    \item \textbf{Em dash (\-/\-/\-/)} is used as a parenthetical break \--- like this \--- in a sentence.
\end{itemize}

\subsection{Quotations}

To ensure proper quotation formatting across languages and document styles, the \texttt{csquotes} package should be used. It provides the \verb|\enquote| command, which automatically adapts to the chosen language. For example: \enquote{This is a correctly formatted quote.} A longer quotation can be set as a block:
\blockquote{This is a block quotation. It is useful for longer passages that should be visually separated from the main text, and it adapts to the document language automatically.}

\subsection{The \texttt{glossaries-extra} Package}

The \texttt{glossaries-extra} package is useful for defining glossary entries, acronyms, initialisms, and symbols. Basic glossary and acronym usage:

\begin{itemize}
    \item \verb|\gls{}| on first use expands to the full form: \gls{api}.
    \item \verb|\glspl{}| gives the plural: \glspl{api}.
    \item Subsequent uses are abbreviated: \gls{api}, \gls{cpu}, \gls{os}, \gls{dr}, \gls{etc}.
    \item \verb|\glsxtrshort{}| forces the short form: \glsxtrshort{cpu}.
    \item \verb|\newinitialism{}| defines initialisms separately from acronyms, appearing in their own glossary list.
\end{itemize}

Symbol entries defined with the \texttt{symbols} glossary type are listed in the symbols glossary. All glossaries are printed via \verb|\printglossaries|, which is called in the main file after the appendix.

\subsection{Lists}

The \texttt{enumitem} package provides flexible list customisation. The template configures five levels of nested enumeration:

\begin{enumerate}
    \item First level (arabic)
    \begin{enumerate}
        \item Second level (alpha)
        \begin{enumerate}
            \item Third level (roman)
            \begin{enumerate}
                \item Fourth level (Alph)
                \begin{enumerate}
                    \item Fifth level (Roman)
                \end{enumerate}
            \end{enumerate}
        \end{enumerate}
    \end{enumerate}
\end{enumerate}

Inline lists are sometimes useful: \begin{enumerate*}
    \item Something
    \item Something else
    \item Something else entirely
\end{enumerate*}.

\subsection{Miscellaneous}

The \texttt{datetime} package formats dates consistently. The current date is \today. The \verb|\datestring| macro provides a localised label: \datestring.

URLs should be typeset using \verb|\url{}| from the \texttt{xurl} package, which allows line\-/breaking at sensible points: \url{https://www.latex-project.org/help/documentation/}.

The \texttt{comment} package allows entire blocks of content to be suppressed without deleting them:
\begin{comment}
    This content is inside a comment environment and will not appear in the output.
    It can span multiple lines and contain arbitrary LaTeX commands.
\end{comment}
The block above is invisible in the compiled output.

The \texttt{pdfpages} package allows including external PDF files directly into the document using \verb|\includepdf[pages=-]{filename.pdf}|, where \texttt{pages=-} includes all pages. This is useful for appending datasheets, signed forms, or other supplementary documents.
