\section{Math Examples}
This section demonstrates various mathematical features supported by the template. Every package listed under the \enquote{Mathematics} category in \mintinline{latex}{packages.sty} is represented here with a usage example. Some of these examples are purely illustrative and are not necessarily mathematically meaningful.

The \gls{gcd} and \gls{lcm} are fundamental concepts in number theory. The \gls{lcm} of two numbers is the smallest number that is a multiple of both.
\begin{equation*}
    \gcd\pbr{a, b} = \mathit{\glsxtrshort{gcd}}\pbr{a, b}
\end{equation*}
\begin{equation*}
    \lcm\pbr{a, b} = \mathit{\glsxtrshort{lcm}}\pbr{a, b}
\end{equation*}

We begin with examples of custom\-/defined relation symbols:
\begin{equation*}
    x \defeq y + z,
    \quad a \whateq b,
    \quad p \hateq q,
    \quad f \coloneqq g,
    \quad M \meq N,
    \quad U \qmeq V
\end{equation*}

Superscript shorthands for inverse and transpose:
\begin{equation*}
    A\inv,
    \quad A\transpose,
    \quad {A\inv}\transpose
\end{equation*}

The \mintinline{latex}{defn} command is used to introduce a \defn{defined term} in bold italic.

The \mintinline{latex}{amortized} command denotes amortized cost: $\amortized{T}\pbr{n}$.

The number of permutations of $r$ items from a set of $n$ elements is given by:
\begin{equation*}
    \perm{n}{r} = \frac{n!}{\pbr{n - r}!}
\end{equation*}

The number of combinations (binomial coefficient) is:
\begin{equation*}
    \choose{n}{r} = \comb{n}{r} = \frac{n!}{r!\pbr{n - r}!}
\end{equation*}

We can also use floor and ceiling functions:
\begin{equation*}
    \ceil{1.2} = 2, \qquad \floor{2.2} = 2
\end{equation*}

The \mintinline{latex}{narrowarray} and \mintinline{latex}{normalarray} commands adjust column spacing in arrays:
\begin{equation*}
    \narrowarray{0.5}
    \begin{pmatrix}
        a & b \\
        c & d
    \end{pmatrix}
    \quad \text{vs.} \quad
    \normalarray
    \begin{pmatrix}
        a & b \\
        c & d
    \end{pmatrix}
\end{equation*}

And comma\-/separated matrix environment:
\begin{equation*}
    \begin{pvector}
        a & b & \ldots & z
    \end{pvector}
\end{equation*}

And auto delimiters:
\begin{equation*}
    a + \abr{b + \abr{c + \abr{d + \abr{e + \abr{f + \abr{g + h} + \abr{i + j} + k} + l} + m} + n} + o} + p
\end{equation*}

\subsection{Expected Value and Variance}

Expected value and variance:
\begin{equation*}
    \expect{X} = \sum_{x \in \mathcal{X}} x \cdot \prob{X = x}
\end{equation*}
\begin{equation*}
    \Var\pbr{X} = \sigmasq\pbr{X} = \expect{\abr{X - \expect{X}}\sq} = \expectsq{X} - \expect{X}\sq
\end{equation*}

\subsection{Equations}

An equation with a label and number, demonstrating \mintinline{latex}{amsmath}:
\begin{equation}\label{eq:euler}
    e^{i\pi} + 1 = 0
\end{equation}

The custom \mintinline{latex}{inequality} environment assigns a label like an equation:
\begin{inequality}\label{ineq:example}
    n\sq \geq n \quad \text{for all } n \geq 1
\end{inequality}

The custom \mintinline{latex}{invariant} environment also assigns a label like an equation:
\begin{invariant}\label{inv:example}
    n\sq \geq n \quad \text{for all } n \geq 1
\end{invariant}

\cref{eq:euler,ineq:example,inv:example}. Equations, partially to demonstrate the package \mintinline{latex}{glossaries-extra}:
\begin{equation*}
    E = m\gls{c}\sq
\end{equation*}

A piecewise\-/defined function using \mintinline{latex}{cif} and \mintinline{latex}{cotherwise}:
\begin{equation*}
    f\pbr{x} = \begin{cases}
        x\sq & \cif{x \geq 0} \\
        -x   & \cotherwise
    \end{cases}
\end{equation*}

We also use \mintinline{latex}{mathclap} to fix spacing in subscripts:
\begin{equation*}
    \sum_{1 \leq i < j \leq n} a_{ij}
    \qquad \text{vs} \qquad
    \sum_{\mathclap{1 \leq i < j \leq n}} a_{ij}
\end{equation*}

Quotient group notation using \mintinline{latex}{faktor}:
\begin{equation*}
    \faktor{G}{N}, \quad \faktor{\ZZ}{n\ZZ}
\end{equation*}

\subsection{Miscellaneous}

Bold vector notation using \mintinline{latex}{bm}:
\begin{equation*}
    \bm{F} = m \bm{a}
\end{equation*}

For matrices and vectors, use the respective commands \mintinline{latex}{\mat} and \mintinline{latex}{\vect} for ease of switching notation, which depends on the setting. Here, \mintinline{latex}{\mat} produces \mintinline{latex}{\mathbf}, while \mintinline{latex}{\vect} produces \mintinline{latex}{\bm}:
\begin{equation*}
    \mat{C}\vect{u} = \lambda \vect{u}
\end{equation*}

Cancellation of terms using \mintinline{latex}{cancel}:
\begin{equation*}
    \frac{\cancel{a} \cdot b}{\cancel{a}} = b
\end{equation*}

The \mintinline{latex}{xfrac} package provides better inline fractions: $\sfrac{a}{b}$ is cleaner than $\frac{a}{b}$ or $a/b$.

\begin{theorem}[Extreme Value Theorem]
    Let $f$ be a continuous function on a closed interval $\intcc{a}{b}$. Then $f$ attains both its maximum and minimum on $\intcc{a}{b}$.
\end{theorem}
\begin{proof}
    By the Extreme Value Theorem, a continuous function on a compact set attains its bounds.
\end{proof}
\begin{proofidea}
    Compactness of $\intcc{a}{b}$ under continuity forces boundedness and attainment.
\end{proofidea}
\begin{proofsketch}
    Cover $\intcc{a}{b}$ by open sets, extract a finite subcover, and simply apply continuity.
\end{proofsketch}
